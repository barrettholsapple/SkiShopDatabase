**Project Overview:**

This project is a web-based rental management system designed to model and improve customer and equipment rental workflows for a ski and snowboard shop. The goal was to replace fragmented, paper-based tracking with a structured, searchable database and a simple employee-facing interface.

The system focuses on organizing customer records, rental history, and equipment inventory in a way that improves accuracy, efficiency, and accessibility for staff.


**Motivation:**

While working at a local ski shop, I noticed that customer and rental information was primarily stored in paper files and disconnected systems, making it difficult to retrieve records quickly or identify trends. This project was created to explore how a relational database and lightweight web interface could streamline rental operations and reduce data-entry errors.
